How it works

This application uses Google's Speech-to-Text API to transcribe audio and video files. The backend is built with FastAPI and the frontend is built with HTML, CSS, and JavaScript.

To handle large files, the selected file is broken into 4MB chunks directly in your browser. Each chunk is uploaded sequentially to the backend. Once all chunks are received, the backend assembles them back into a single file and sends it directly to Google's Speech-to-Text API for transcription. Google handles all audio format processing automatically, so no additional audio processing tools are needed. The final transcript is then displayed here.

Supported Formats

Audio: MP3, WAV, FLAC, M4A, AAC, OGG, and more

Video: MP4, MOV, AVI, MKV, WebM, FLV, and more

Google Speech-to-Text automatically extracts audio from video files and supports most common formats. The service automatically detects the format and processes it accordingly.
