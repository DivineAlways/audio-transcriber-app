How it works

This application uses Google's Speech-to-Text API to transcribe audio files. The backend is built with FastAPI and the frontend is built with HTML, CSS, and JavaScript.

When you upload an audio file, it is sent to the FastAPI backend. The backend then splits the audio into smaller chunks and sends them to the Google Speech-to-Text API for transcription. The transcriptions are then combined and sent back to the frontend to be displayed.
